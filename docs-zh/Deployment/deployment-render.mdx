---
title: 在 Render 上部署 Langflow
slug: /deployment-render
---

本指南解释如何在 [Render](https://render.com/) 上部署 Langflow，Render 是一个用于部署 Web 应用程序和 API 的云平台。

:::note
Langflow 需要至少 2 GB 的 RAM 才能运行，因此它使用 **标准** 的 Render 实例。这可能需要信用卡。在继续之前，请查看 [Render 的定价](https://render.com/pricing)。
:::

1. 点击以下按钮转到 Render：

   [![Deploy to Render](/logos/render-deploy.svg)](https://render.com/deploy?repo=https%3A%2F%2Fgithub.com%2Flangflow-ai%2Flangflow%2Ftree%2Fdev)

2. 输入蓝图名称，然后为您的 `render.yaml` 文件选择分支。

3. 点击 **Deploy Blueprint**。

等待部署完成。

您的 Langflow 实例现在已准备就绪。
