本指南解释如何在 [HuggingFace Spaces](https://huggingface.co/spaces/) 上部署 Langflow。

1. 转到 [Langflow Space](https://huggingface.co/spaces/Langflow/Langflow?duplicate=true)。

2. 点击 **Duplicate Space**。
3. 在配置对话框中，执行以下操作：
   - 为您的 Space 输入名称。
   - 选择公开或私有可见性。
   - 点击 **Duplicate Space**

   ![Hugging Face deployment dialog](/img/hugging-face-deployment.png)

等待设置完成。您将自动重定向到新的 Space。

您的 Langflow 实例现在已准备就绪。
