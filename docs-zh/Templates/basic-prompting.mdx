此流程是理解 Langflow 的起点。
该流程演示了如何与 LLM 聊天，以及修改提示如何影响您的结果。

提示作为大型语言模型 (LLM) 的输入，充当人类指令和计算任务之间的接口。

通过在提示中向 LLM 提交自然语言请求，您可以获得答案、生成文本并解决问题。

## 先决条件

- [正在运行的 Langflow 实例](/get-started-installation)
- [OpenAI API 密钥](https://platform.openai.com/)

## 创建基础提示流程

1. 从 Langflow 仪表板，点击**新建流程**。

2. 选择**基础提示**。

3. **基础提示**流程被创建。

    ![Basic prompting flow](/img/starter-flow-basic-prompting.png)

    此流程允许您与 **OpenAI 模型**组件聊天。
    该模型将根据**提示**组件中构建的提示进行响应。

4. 要检查**模板**，在**提示**组件中，点击**模板**字段。
    ```text
    Answer the user as if you were a GenAI expert, enthusiastic about helping them get started building something fresh.
    ```

5. 要为 **OpenAI** 组件创建环境变量，在 **OpenAI API 密钥**字段中，点击 <Icon name="Globe" aria-hidden="true"/> **地球**按钮，然后点击**添加新变量**。

	1. 在**变量名称**字段中，输入 `openai_api_key`。
	2. 在**值**字段中，粘贴您的 OpenAI API 密钥（`sk-...`）。
	3. 点击**保存变量**。

## 运行基础提示流程

1. 点击**操场**按钮。
2. 输入一条消息并按 Enter 键。机器人应该以明显的海盗方式响应！

## 修改提示以获得不同结果

1. 要修改您的提示结果，在**提示**组件中，点击**模板**字段。**编辑提示**窗口打开。
2. 将现有提示更改为不同的角色，例如 `Answer the user as if you were Hermione Granger.`
3. 再次运行工作流并注意提示如何改变模型的响应。
