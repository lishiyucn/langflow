此流程演示了如何向[提示](/components-prompts)组件添加额外的 `{references}` 输入，以捐获来自 [URL](/components-data#url) 和[解析器](/components-processing#parser)组件的输入。

[语言模型](/components-models)组件使用此输入根据[提示](/components-prompts)组件接收到的 `{instructions}` 生成博客文章。

## 先决条件

- [正在运行的 Langflow 实例](/get-started-installation)
- [OpenAI API 密钥](https://platform.openai.com/)

## 创建博客写作器流程

1. 从 Langflow 仪表板，点击**新建流程**。
2. 选择**博客写作器**。
3. **博客写作器**流程被创建。

    ![Blog writer starter flow](/img/starter-flow-blog-writer.png)

    此流程使用**提示**、**语言模型**和**聊天输出**组件创建了一个博客文章生成器，并通过 **URL** 和**文本输入**组件的参考内容和指令进行增强。

    **URL** 组件从一个或多个网络链接中提取原始文本和元数据。
    **解析器**组件将来自 **URL** 组件的数据转换为纯文本以供给提示。

    要检查流程的提示，请点击**提示**组件的**模板**字段。

    ```text
    Reference 1:

    {references}

    ---

    {instructions}

    Blog:
    ```

    `{instructions}` 值来自**文本输入**组件，一个或多个 `{references}` 来自从 **URL** 组件解析的 URL 列表。


### 运行博客写作器流程

1. 将您的 **OpenAI API 密钥**添加到**语言模型**模型组件。
	可选地，为 **OpenAI API 密钥**创建一个[全局变量](/configuration-global-variables)。

	1. 在 **OpenAI API 密钥**字段中，点击 <Icon name="Globe" aria-hidden="True" /> **地球**，然后点击**添加新变量**。
	2. 在**变量名称**字段中，输入 `openai_api_key`。
	3. 在**值**字段中，粘贴您的 OpenAI API 密钥（`sk-...`）。
	4. 点击**保存变量**。

2. 要运行流程，请点击 <Icon name="Play" aria-hidden="true"/> **操场**，然后点击**运行流程**。
将生成一篇关于 Langflow 的博客文章，内容来源于 `langflow.org` 和 `docs.langflow.org`。
3. 要写关于不同主题的内容，请更改 **URL** 组件中的值，并调整**文本输入**组件中的指令。再试一次，看看 LLM 构建了什么。
