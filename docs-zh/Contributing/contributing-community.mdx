有多种方式可以与 Langflow 社区互动并深入了解 Langflow 代码库。

## 加入 Langflow Discord 服务器

加入 [Langflow Discord Server](https://discord.gg/EqksyE2EX9) 来提问和展示您的项目。

## 在 X 上关注 Langflow

在 X 上关注 [@langflow_ai](https://twitter.com/langflow_ai) 获取关于 Langflow 的最新消息。

## 在 GitHub 上为 Langflow 点星

如果您喜欢 Langflow，可以为 [Langflow GitHub repository](https://github.com/langflow-ai/langflow) 点星。
星标帮助其他用户更容易找到 Langflow，并快速了解其他用户认为它很有用。

由于 Langflow 是一个开源项目，代码库的可见度越高，项目就越有可能吸引[贡献者](/contributing-how-to-contribute)。

## 关注 GitHub 代码库

您可以关注 [Langflow GitHub repository](https://github.com/langflow-ai/langflow) 以获得新版本发布和其他代码库活动的通知。

如果只想获得发布通知，请选择 **Releases only**。

如果您选择 **Watching**，您将收到关于新版本发布以及问题、讨论和拉取请求的通知（如果您对这些活动感兴趣的话）。
有关自定义代码库通知的信息，请参阅 [GitHub documentation on repository subscriptions](https://docs.github.com/en/account-and-profile/managing-subscriptions-and-notifications-on-github/managing-subscriptions-for-activity-on-github/viewing-your-subscriptions)。

## 通过 GitHub 请求功能增强和获取帮助

您也可以通过 GitHub 代码库提交功能请求和获取 Langflow 的帮助。
更多信息请参阅 [Get help and request enhancements](/contributing-github-issues)。
