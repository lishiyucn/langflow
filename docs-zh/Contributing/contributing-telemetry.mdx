Langflow 使用匿名遥测来收集基本的使用统计数据，以增强功能和用户体验。这些数据帮助我们识别热门功能和需要改进的领域，并确保开发工作与您的需求保持一致。

我们尊重您的隐私并致力于保护您的数据。我们不收集任何个人信息或敏感数据。所有遥测数据都是匿名的，仅用于改进 Langflow。

## 退出遥测

要退出遥测，请在运行 Langflow 之前将 `LANGFLOW_DO_NOT_TRACK` 或 `DO_NOT_TRACK` 环境变量设置为 `true`。这将禁用遥测数据收集。

## Langflow 收集的数据

### 运行 {#2d427dca4f0148ae867997f6789e8bfb}

- **IsWebhook**: 指示操作是否通过 webhook 触发。
- **Seconds**: 操作持续时间（以秒为单位），提供性能洞察。
- **Success**: 布尔值，指示操作是否成功，有助于识别潜在错误或问题。
- **ErrorMessage**: 如果操作不成功，提供错误消息详情，有助于故障排除和改进。

### 关闭 {#081e4bd4faec430fb05b657026d1a69c}

- **Time Running**: 关闭前的总运行时间，有助于了解应用程序生命周期和优化正常运行时间。

### 版本 {#dc09f6aba6c64c7b8dad3d86a7cba6d6}

- **Version**: 使用的 Langflow 特定版本，有助于跟踪功能采用情况和兼容性。
- **Platform**: 主机的操作系统，有助于我们专注于对 Windows、macOS 和 Linux 等热门平台的支持。
- **Python**: 使用的 Python 版本，有助于维护对各种 Python 版本的兼容性和支持。
- **Arch**: 系统架构（例如 x86、ARM），有助于为不同硬件优化我们的软件。
- **AutoLogin**: 指示是否启用自动登录功能，反映用户偏好设置。
- **CacheType**: 使用的缓存机制类型，影响性能和效率。
- **BackendOnly**: 布尔值，指示您是否在仅后端模式下运行 Langflow，有助于了解部署配置。

### 游乐场 {#ae6c3859f612441db3c15a7155e9f920}

- **Seconds**: 游乐场执行的持续时间（以秒为单位），提供测试或实验阶段的性能洞察。
- **ComponentCount**: 游乐场中使用的组件数量，有助于了解复杂性和使用模式。
- **Success**: 游乐场操作的成功状态，有助于识别实验功能的稳定性。

### 组件 {#630728d6654c40a6b8901459a4bc3a4e}

- **Name**: 识别组件，提供关于哪些组件使用最多或容易出现问题的数据。
- **Seconds**: 组件执行所用时间，提供性能指标。
- **Success**: 组件是否成功运行，有助于质量控制。
- **ErrorMessage**: 遇到的任何错误详情，对调试和改进至关重要。
