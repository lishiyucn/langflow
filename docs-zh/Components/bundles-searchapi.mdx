[Bundle](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

本页面描述 **SearchApi** bundle 中可用的组件。

有关更多信息，请参阅 [SearchApi 文档](https://www.searchapi.io/docs/google)。

## SearchApi 网络搜索

此组件调用 SearchApi API 来运行 Google、Bing 和 DuckDuckGo 网络搜索。

它以 [`DataFrame`](/data-types#dataframe) 的形式返回搜索结果列表。

### SearchApi 网络搜索参数

**SearchApi** 组件的一些输入参数在可视化编辑器中默认隐藏。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **Controls** 切换参数。

| 名称 | 类型 | 描述 |
|------|------|-------------|
| engine | String | 输入参数。要使用的搜索引擎。默认值：`google`。 |
| api_key | SecretString | 输入参数。用于 SearchApi 身份验证的 API 密钥。 |
| input_value | String | 输入参数。API 调用的搜索查询或输入。 |
| max_results | Integer | 输入参数。返回的最大搜索结果数量。默认值：`5`。 |
| max_snippet_length | Integer | 输入参数。返回的摘要的最大长度。默认值：`100`。 |
| search_params | Dict | 输入参数。用于自定义请求的其他键值对。 |
