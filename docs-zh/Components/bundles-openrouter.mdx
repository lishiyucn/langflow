[Bundle](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

本页面描述 **OpenRouter** bundle 中可用的组件。

有关 OpenRouter 组件使用的 OpenRouter 功能的更多信息，请参阅 [OpenRouter 文档](https://openrouter.ai/docs)。

## OpenRouter 文本生成

此组件使用 OpenRouter 的统一 API 为来自不同提供商的多个 AI 模型生成文本。

它可以输出 **Model Response** ([`Message`](/data-types#message)) 或 **Language Model** ([`LanguageModel`](/data-types#languagemodel))。

当您想要使用 OpenRouter 模型作为其他 LLM 驱动组件（如 **Language Model** 或 **Smart Function** 组件）的 LLM 时，请使用 **Language Model** 输出。

有关更多信息，请参阅 [**Language Model** 组件](/components-models)。

### OpenRouter 文本生成参数

**OpenRouter** 组件的许多输入参数在可视化编辑器中默认隐藏。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **Controls** 切换参数。

| 名称 | 类型 | 描述 |
|------|------|-------------|
| api_key | SecretString | 输入参数。您的 OpenRouter API 密钥用于身份验证。 |
| site_url | String | 输入参数。您用于 OpenRouter 排名的网站 URL（高级）。 |
| app_name | String | 输入参数。您用于 OpenRouter 排名的应用程序名称（高级）。 |
| provider | String | 输入参数。要使用的 AI 模型提供商。 |
| model_name | String | 输入参数。用于聊天完成的特定模型。 |
| temperature | Float | 输入参数。控制输出中的随机性。范围：[0.0, 2.0]。默认值：0.7。 |
| max_tokens | Integer | 输入参数。要生成的最大 token 数量（高级）。 |
