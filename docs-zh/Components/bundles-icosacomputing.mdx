[组件包](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

**Icosa Computing** 组件需要访问 Icosa Computing 服务。
有关更多信息和请求访问权限，请参阅 [Icosa Computing 站点](https://www.icosacomputing.com/)。

## Combinatorial Reasoner

**Combinatorial Reasoner** 组件在输入上运行 Icosa 的组合推理 (CR) 流程，以创建包含嵌入推理的优化提示。

### Combinatorial Reasoner 参数

| 名称 | 类型 | 描述 |
|------|------|-------------|
| prompt | String | 输入参数。要运行 CR 的输入。 |
| openai_api_key | SecretString | 输入参数。用于对 OpenAI 进行身份验证的 OpenAI API 密钥，用于推理生成。 |
| model_name | String | 输入参数。用于推理生成的 OpenAI LLM。 |
| username | String | 输入参数。用于 Icosa API 身份验证的用户名。 |
| password | SecretString | 输入参数。用于 Icosa API 身份验证的密码。 |

### Combinatorial Reasoner 输出

**Combinatorial Reasoner** 组件输出以下之一：

* **优化提示 (`optimized_prompt`)**：[一个 `Message`](/data-types#message) 对象，包含嵌入推理的优化提示。
* **选定推理 (`reasons`)**：一个 [`Data`](/data-types#data) 对象，包含字符串列表，其中每个字符串都是被选中并嵌入优化提示中的推理。

您可以在组件输出端口附近切换输出类型。
