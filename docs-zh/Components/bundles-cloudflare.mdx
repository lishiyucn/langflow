[Bundle](/components-bundle-components) 包含支持 Langflow 特定第三方集成的自定义组件。

本页描述了 **Cloudflare** bundle 中可用的组件。

## Cloudflare Workers AI 嵌入模型

**Cloudflare Workers AI 嵌入模型**组件使用 [Cloudflare Workers AI 模型](https://developers.cloudflare.com/workers-ai/) 生成嵌入。

有关在流程中使用嵌入模型组件的更多信息，请参阅[**嵌入模型**组件](/components-embedding-models)。

### Cloudflare Workers AI 嵌入模型参数

某些 **Cloudflare Workers AI 嵌入模型**组件输入参数在可视化编辑器中默认隐藏。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的<Icon name="SlidersHorizontal" aria-hidden="true"/> **控制**切换参数。

| 名称 | 显示名称 | 信息 |
|------|--------------|------|
| account_id | Cloudflare 账户 ID | 输入参数。您的 [Cloudflare 账户 ID](https://developers.cloudflare.com/fundamentals/setup/find-account-and-zone-ids/#find-account-id-workers-and-pages)。 |
| api_token | Cloudflare API 令牌 | 输入参数。您的 [Cloudflare API 令牌](https://developers.cloudflare.com/fundamentals/api/get-started/create-token/)。 |
| model_name | 模型名称 | 输入参数。用于嵌入生成的[支持的模型](https://developers.cloudflare.com/workers-ai/models/#text-embeddings)。 |
| strip_new_lines | 去除换行符 | 输入参数。是否从输入文本中去除换行符。 |
| batch_size | 批处理大小 | 输入参数。每批嵌入的文本数量。 |
| api_base_url | Cloudflare API 基础 URL | 输入参数。Cloudflare API 的基础 URL。 |
| headers | 请求头 | 输入参数。嵌入生成 API 请求的附加请求头。 |
