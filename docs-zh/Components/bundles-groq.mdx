[Bundle组件包](/components-bundle-components) 包含支持 Langflow 特定第三方集成的自定义组件。

本页描述了 **Groq** bundle 中可用的组件。

更多信息请参阅 [Groq 文档](https://groq.com/)。

## Groq 文本生成

该组件使用 Groq 的语言模型生成文本。

它可以输出 **模型响应** ([`Message`](/data-types#message)) 或 **语言模型** ([`LanguageModel`](/data-types#languagemodel))。
具体来说，**语言模型** 输出是根据组件参数配置的 [`ChatGroq`](https://python.langchain.com/docs/integrations/chat/groq/) 实例。

当您想要使用 Groq 模型作为其他 LLM 驱动组件（如 **Language Model** 或 **Smart Function** 组件）的 LLM 时，请使用 **语言模型** 输出。

更多信息请参阅 [**语言模型** 组件](/components-models)。

### Groq 文本生成参数

| 名称 | 类型 | 描述 |
|------|------|-------------|
| groq_api_key | SecretString | 您的 Groq API 密钥。 |
| groq_api_base | String | API 请求的基础 URL 路径。默认：`https://api.groq.com`。 |
| max_tokens | Integer | 要生成的最大令牌数。 |
| temperature | Float | 控制输出中的随机性。范围：`[0.0, 1.0]`。默认：`0.1`。 |
| n | Integer | 为每个提示生成的聊天完成数量。 |
| model_name | String | 要使用的 Groq 模型名称。选项在输入您的 API 密钥和 URL 后从 Groq API 动态获取。要刷新模型列表，请点击 <Icon name="RefreshCw" aria-hidden="true"/> **刷新**。 |
| tool_mode_enabled | Boolean | 如果启用，组件仅显示与工具兼容的模型。 |
