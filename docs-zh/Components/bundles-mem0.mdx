[组件包](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

本页面介绍 **Mem0** 组件包中可用的组件。

## Mem0 Chat Memory

**Mem0 Chat Memory** 组件使用 Mem0 内存存储检索和存储聊天消息。

### Mem0 Chat Memory 参数

许多 **Mem0 Chat Memory** 组件输入参数在可视化编辑器中默认为隐藏状态。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **控件** 来切换参数。

| 名称 | 显示名称 | 信息 |
|------|--------------|------|
| mem0_config | Mem0 配置 | 输入参数。用于初始化 Mem0 内存实例的配置字典。 |
| ingest_message | 要摄取的消息 | 输入参数。要摄取到 Mem0 内存中的消息内容。 |
| existing_memory | 现有内存实例 | 输入参数。可选的现有 Mem0 内存实例。 |
| user_id | 用户 ID | 输入参数。与消息关联的用户标识符。 |
| search_query | 搜索查询 | 输入参数。用于在 Mem0 中搜索相关记忆的输入文本。 |
| mem0_api_key | Mem0 API 密钥 | 输入参数。Mem0 平台的 API 密钥。留空以使用本地版本。 |
| metadata | 元数据 | 输入参数。与摄取消息关联的额外元数据。 |
| openai_api_key | OpenAI API 密钥 | 输入参数。OpenAI 的 API 密钥。在使用 OpenAI 嵌入而未提供配置时必需。 |

### Mem0 Chat Memory 输出

**Mem0 Chat Memory** 组件可以输出 **Mem0 内存**（[`Memory`](/data-types#memory)）或 **搜索结果**（[`Data`](/data-types#data)）。
您可以在组件输出端口附近选择输出类型。

使用 **Mem0 Chat Memory** 与 [**消息历史** 组件](/components-helpers#message-history)进行内存存储和检索操作。

使用 **搜索结果** 输出基于搜索查询检索特定记忆。
