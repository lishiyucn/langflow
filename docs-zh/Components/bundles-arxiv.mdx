[组件包](/components-bundle-components)包含支持 Langflow 与特定第三方集成的自定义组件。

本页面描述了 **arXiv** 组件包中可用的组件。

## arXiv 搜索

该组件搜索并从 [arXiv.org](https://arXiv.org) 检索论文。

它以 [`DataFrame`](/data-types#dataframe) 的形式返回搜索结果列表。

### arXiv 搜索参数

| 名称 | 类型 | 描述 |
|------|------|-------------|
| search_query | String | 输入参数。arXiv 论文的搜索查询。例如，`quantum computing`。 |
| search_type | String | 输入参数。要搜索的字段。 |
| max_results | Integer | 输入参数。要返回的最大结果数。 |
