[Bundle](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

本页面描述 **Redis** bundle 中可用的组件。

## Redis Chat Memory

**Redis Chat Memory** 组件使用 Redis 内存存储检索和存储聊天消息。

聊天记忆作为 [`Memory`](/data-types#memory) 数据类型在内存存储组件之间传递。

有关在流中使用外部聊天记忆的更多信息，请参阅 [**Message History** 组件](/components-helpers#message-history)。

### Redis Chat Memory 参数

**Redis Chat Memory** 组件的许多输入参数在可视化编辑器中默认隐藏。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **Controls** 切换参数。

| 名称 | 显示名称 | 信息 |
|------|--------------|------|
| host | hostname | 输入参数。IP 地址或主机名。 |
| port | port | 输入参数。Redis 端口号。 |
| database | database | 输入参数。Redis 数据库。 |
| username | Username | 输入参数。Redis 用户名。 |
| password | Password | 输入参数。用户名的密码。 |
| key_prefix | Key prefix | 输入参数。键前缀。 |
| session_id | Session ID | 输入参数。消息的唯一会话标识符。 |
