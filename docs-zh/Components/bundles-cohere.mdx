[Bundle](/components-bundle-components) 包含支持 Langflow 特定第三方集成的自定义组件。

本页描述了 **Cohere** bundle 中可用的组件。

有关 Cohere 组件使用的 Cohere 功能和特性的更多信息，请参阅 [Cohere 文档](https://cohere.ai/)。

### Cohere 文本生成

此组件使用 Cohere 的语言模型生成文本。

它可以输出 **模型响应** ([`Message`](/data-types#message)) 或 **语言模型** ([`LanguageModel`](/data-types#languagemodel))。

当您想要将 Cohere 模型用作其他 LLM 驱动组件（如 **语言模型**或 **智能函数**组件）的 LLM 时，请使用 **语言模型**输出。

有关更多信息，请参阅[**语言模型**组件](/components-models)。

### Cohere 文本生成参数

许多 **Cohere** 组件输入参数在可视化编辑器中默认隐藏。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的<Icon name="SlidersHorizontal" aria-hidden="true"/> **控制**切换参数。

| 名称 | 类型 | 描述 |
|------|------|-------------|
| Cohere API Key | SecretString | 输入参数。您的 Cohere API 密钥。 |
| Max Tokens | Integer | 输入参数。指定要生成的最大令牌数。默认为 `256`。 |
| Temperature | Float | 输入参数。指定采样温度。默认为 `0.75`。 |
| Input Value | String | 输入参数。指定文本生成的输入文本。 |

### Cohere 嵌入模型

**Cohere 嵌入模型**组件用于从 Cohere 加载嵌入模型。

有关在流程中使用嵌入模型组件的更多信息，请参阅[**嵌入模型**组件](/components-embedding-models)。

### Cohere 嵌入模型参数

| 名称 | 类型 | 描述 |
|------|------|-------------|
| cohere_api_key | String | 输入参数。与 Cohere 服务进行身份验证所需的 API 密钥。 |
| model | String | 输入参数。用于嵌入文本文档和执行查询的语言模型。默认：`embed-english-v2.0`。 |
| truncate | Boolean | 输入参数。是否截断输入文本以适应模型的约束。默认：`False`。 |
