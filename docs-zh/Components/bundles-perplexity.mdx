[Bundle](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

本页面描述 **Perplexity** bundle 中可用的组件。

有关 Perplexity 组件使用的 Perplexity 功能的更多信息，请参阅 [Perplexity 文档](https://perplexity.ai/)。

## Perplexity 文本生成

此组件使用 Perplexity 的语言模型生成文本。

它可以输出 **Model Response** ([`Message`](/data-types#message)) 或 **Language Model** ([`LanguageModel`](/data-types#languagemodel))。

当您想要使用 Perplexity 模型作为其他 LLM 驱动组件（如 **Language Model** 或 **Smart Function** 组件）的 LLM 时，请使用 **Language Model** 输出。

有关更多信息，请参阅 [**Language Model** 组件](/components-models)。

### Perplexity 文本生成参数

**Perplexity** 组件的许多输入参数在可视化编辑器中默认隐藏。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **Controls** 切换参数。

| 名称 | 类型 | 描述 |
|------|------|-------------|
| model_name | String | 输入参数。要使用的 Perplexity 模型名称。选项包括各种 Llama 3.1 模型。 |
| max_output_tokens | Integer | 输入参数。要生成的最大 token 数量。 |
| api_key | SecretString | 输入参数。用于身份验证的 Perplexity API 密钥。 |
| temperature | Float | 输入参数。控制输出中的随机性。默认值：0.75。 |
| top_p | Float | 输入参数。采样时要考虑的 token 的最大累积概率（高级）。 |
| n | Integer | 输入参数。为每个提示生成的聊天完成数量（高级）。 |
| top_k | Integer | 输入参数。top-k 采样要考虑的顶部 token 数量。必须为正数（高级）。 |
