[组件包](/components-bundle-components)包含支持 Langflow 与特定第三方集成的自定义组件。

本页面描述了 **Anthropic** 组件包中可用的组件。

有关 Anthropic 组件使用的 Anthropic 功能和功能性的更多信息，请参阅 [Anthropic 文档](https://docs.anthropic.com/en/docs/intro)。

## Anthropic 文本生成

**Anthropic** 组件使用 Anthropic Chat 和语言模型（如 Claude）生成文本。

它可以输出**模型响应** ([`Message`](/data-types#message)) 或**语言模型** ([`LanguageModel`](/data-types#languagemodel))。
具体来说，**语言模型**输出是根据组件参数配置的 [`ChatAnthropic`](https://python.langchain.com/docs/integrations/chat/anthropic/) 实例。

当您希望将 Anthropic 模型用作其他 LLM 驱动组件（例如**语言模型**或**智能函数**组件）的 LLM 时，请使用**语言模型**输出。

有关更多信息，请参阅[**语言模型**组件](/components-models)。

### Anthropic 文本生成参数

默认情况下，许多 **Anthropic** 组件输入参数在可视化编辑器中是隐藏的。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **控制**来切换参数。

| 名称 | 类型 | 描述 |
|------|------|-------------|
| max_tokens | Integer | 输入参数。要生成的最大 token 数。设置为 0 表示无限 token。默认值：`4096`。 |
| model | String | 输入参数。要使用的 Anthropic 模型名称。选项包括各种 Claude 3 模型。 |
| anthropic_api_key | SecretString | 输入参数。您用于身份验证的 Anthropic API 密钥。 |
| temperature | Float | 输入参数。控制输出中的随机性。默认值：`0.1`。 |
| anthropic_api_url | String | 输入参数。Anthropic API 的端点。如果未指定，默认为 `https://api.anthropic.com`（高级）。 |
| prefill | String | 输入参数。用于指导模型响应的预填充文本（高级）。 |
