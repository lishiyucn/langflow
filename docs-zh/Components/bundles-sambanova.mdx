[Bundle](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

本页面描述 **SambaNova** bundle 中可用的组件。

有关 SambaNova 组件使用的 SambaNova 功能的更多信息，请参阅 [SambaNova Cloud 文档](https://cloud.sambanova.ai/)。

## SambaNova 文本生成

此组件使用 SambaNova LLM 生成文本。

它可以输出 **Model Response** ([`Message`](/data-types#message)) 或 **Language Model** ([`LanguageModel`](/data-types#languagemodel))。

当您想要使用 SambaNova 模型作为其他 LLM 驱动组件（如 **Language Model** 或 **Smart Function** 组件）的 LLM 时，请使用 **Language Model** 输出。

有关更多信息，请参阅 [**Language Model** 组件](/components-models)。

### SambaNova 文本生成参数

**SambaNova** 组件的许多输入参数在可视化编辑器中默认隐藏。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **Controls** 切换参数。

| 名称 | 类型 | 描述 |
|------|------|-------------|
| sambanova_url | String | 输入参数。API 请求的基础 URL 路径。默认值：`https://api.sambanova.ai/v1/chat/completions`。 |
| sambanova_api_key | SecretString | 输入参数。您的 SambaNova API 密钥。 |
| model_name | String | 输入参数。要使用的 SambaNova 模型名称。选项包括各种 Llama 模型。 |
| max_tokens | Integer | 输入参数。要生成的最大 token 数量。设置为 0 表示无限制 token。 |
| temperature | Float | 输入参数。控制输出中的随机性。范围：[0.0, 1.0]。默认值：0.07。 |
