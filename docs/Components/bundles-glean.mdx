[Bundle](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

本页面描述了 **Glean** bundle 中可用的组件。

## Glean Search API

此组件允许您调用 Glean Search API。

它返回搜索结果列表作为 [`DataFrame`](/data-types#dataframe)。

### Glean Search API 参数

**Glean Search API** 组件的一些输入参数在可视化编辑器中默认隐藏。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **Controls** 切换参数。

| 名称 | 类型 | 描述 |
|------|------|-------------|
| glean_api_url | String | 输入参数。Glean API 的 URL。 |
| glean_access_token | SecretString | 输入参数。用于 Glean API 身份验证的访问令牌。 |
| query | String | 输入参数。搜索查询输入。 |
| page_size | Integer | 输入参数。每页结果数量。默认：10。 |
| request_options | Dict | 输入参数。API 请求的附加选项。 |
