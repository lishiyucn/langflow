[Bundle](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

本页面描述了 **Exa** bundle 中可用的组件。

## Exa Search

此组件为 [Langflow 代理](/agents) 或 [MCP 客户端](/mcp-client) 提供 [Exa Search](https://exa.ai/) 工具包，用于搜索和内容检索。

输出专门为 [`Tools`](/data-types#tool)。

### Exa Search 参数

| 名称 | 类型 | 描述 |
|------|------|-------------|
| Exa Search API Key (`metaphor_api_key`) | SecretString | 输入参数。Exa Search 的 API 密钥。 |
| Use Autoprompt (`use_autoprompt`) | Boolean | 输入参数。是否使用自动提示功能。默认：true。 |
| Search Number of Results (`search_num_results`) | Integer | 输入参数。搜索返回的结果数量。默认：5。 |
| Similar Number of Results (`similar_num_results`) | Integer | 输入参数。返回的相似结果数量。默认：5。 |
