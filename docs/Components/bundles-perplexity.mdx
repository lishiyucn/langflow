[组件包](/components-bundle-components)包含支持与 Langflow 特定第三方集成的自定义组件。

本页介绍了 **Perplexity** 组件包中可用的组件。

有关 Perplexity 组件所使用的 Perplexity 功能和特性的更多信息，请参阅 [Perplexity 文档](https://perplexity.ai/)。

## Perplexity 文本生成

该组件使用 Perplexity 的语言模型生成文本。

它可以输出**模型响应** ([`Message`](/data-types#message)) 或**语言模型** ([`LanguageModel`](/data-types#languagemodel))。

当您想要将 Perplexity 模型用作另一个LLM驱动组件（如**语言模型**或**智能函数**组件）的 LLM 时，请使用**语言模型**输出。

有关更多信息，请参阅[**语言模型**组件](/components-models)。

### Perplexity 文本生成参数

许多 **Perplexity** 组件输入参数在可视化编辑器中默认是隐藏的。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **控制** 来切换参数。

| 名称 | 类型 | 描述 |
|------|------|-------------|
| model_name | String | 输入参数。要使用的 Perplexity 模型名称。选项包括各种 Llama 3.1 模型。 |
| max_output_tokens | Integer | 输入参数。要生成的最大令牌数。 |
| api_key | SecretString | 输入参数。用于身份验证的 Perplexity API 密钥。 |
| temperature | Float | 输入参数。控制输出的随机性。默认值：0.75。 |
| top_p | Float | 输入参数。采样时考虑的令牌的最大累积概率（高级选项）。 |
| n | Integer | 输入参数。为每个提示生成的聊天完成数量（高级选项）。 |
| top_k | Integer | 输入参数。用于 top-k 采样的顶级令牌数量。必须为正数（高级选项）。 |
