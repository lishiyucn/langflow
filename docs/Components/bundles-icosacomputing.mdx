[Bundle](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

**Icosa Computing** 组件需要访问 Icosa Computing 服务。
有关更多信息和请求访问权限，请参阅 [Icosa Computing 网站](https://www.icosacomputing.com/)。

## Combinatorial Reasoner

**Combinatorial Reasoner** 组件在输入上运行 Icosa 的组合推理 (CR) 管道，以创建具有嵌入式理由的优化提示。

### Combinatorial Reasoner 参数

| 名称 | 类型 | 描述 |
|------|------|-------------|
| prompt | String | 输入参数。要在其上运行 CR 的输入。 |
| openai_api_key | SecretString | 输入参数。用于 OpenAI 身份验证的 OpenAI API 密钥，用于理由生成。 |
| model_name | String | 输入参数。用于理由生成的 OpenAI LLM。 |
| username | String | 输入参数。用于 Icosa API 身份验证的用户名。 |
| password | SecretString | 输入参数。用于 Icosa API 身份验证的密码。 |

### Combinatorial Reasoner 输出

**Combinatorial Reasoner** 组件输出以下之一：

* **Optimized Prompt (`optimized_prompt`)**：包含具有嵌入式理由的优化提示的 [`Message`](/data-types#message) 对象。
* **Selected Reasons (`reasons`)**：包含字符串列表的 [`Data`](/data-types#data) 对象，其中每个字符串都是被选中并嵌入到优化提示中的理由。

您可以在组件输出端口附近切换输出类型。
