[Bundle](/components-bundle-components) 包含支持与 Langflow 特定第三方集成的自定义组件。

**LM Studio** bundle 中的组件让您可以使用来自本地或托管 LM Studio 实例的模型。
组件可能需要使用 LM Studio API 密钥进行身份验证。有关 LM Studio 模型、连接和凭据的信息，请参阅 [LM Studio 文档](https://lmstudio.ai/docs)。

## LM Studio 文本生成

**LM Studio** 组件使用 LM Studio 的本地语言模型生成文本。

它可以输出 **Model Response**（[`Message`](/data-types#message)）或 **Language Model**（[`LanguageModel`](/data-types#languagemodel)）。

当您想要使用 LM Studio 模型作为其他 LLM 驱动组件（如 **Language Model** 或 **Smart Function** 组件）的 LLM 时，请使用 **Language Model** 输出。

有关更多信息，请参阅 [**Language Model** 组件](/components-models)。

### LM Studio 文本生成参数

**LM Studio** 组件的许多输入参数在可视化编辑器中默认隐藏。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **Controls** 切换参数。

| 名称 | 类型 | 描述 |
|------|------|-------------|
| base_url | String | 输入参数。LM Studio 运行的 URL。默认：`"http://localhost:1234"`。 |
| api_key | LM Studio API Key | 输入参数。与 LM Studio 身份验证的 API 密钥（如果需要）。 |
| max_tokens | Integer | 输入参数。在响应中生成的最大令牌数。默认：`512`。 |
| temperature | Float | 输入参数。控制输出的随机性。范围：`[0.0, 2.0]`。默认：`0.7`。 |
| top_p | Float | 输入参数。通过核采样控制多样性。范围：`[0.0, 1.0]`。默认：`1.0`。 |
| stop | List[String] | 输入参数。遇到时停止生成的字符串列表。 |
| stream | Boolean | 输入参数。是否流式传输响应。默认：`False`。 |
| presence_penalty | Float | 输入参数。惩罚重复令牌。范围：`[-2.0, 2.0]`。默认：`0.0`。 |
| frequency_penalty | Float | 输入参数。惩罚频繁令牌。范围：`[-2.0, 2.0]`。默认：`0.0`。 |

## LM Studio Embeddings

**LM Studio Embeddings** 组件使用 LM Studio 模型生成嵌入。

有关在流中使用嵌入模型组件的更多信息，请参阅 [**Embedding Model** 组件](/components-embedding-models)。

### LM Studio Embeddings 参数

**LM Studio Embeddings** 组件的许多输入参数在可视化编辑器中默认隐藏。
您可以通过[组件标题菜单](/concepts-components#component-menus)中的 <Icon name="SlidersHorizontal" aria-hidden="true"/> **Controls** 切换参数。

| 名称 | 显示名称 | 信息 |
|------|--------------|------|
| model | Model | 输入参数。用于生成嵌入的 LM Studio 模型。 |
| base_url | LM Studio Base URL | 输入参数。LM Studio API 的基础 URL。 |
| api_key | LM Studio API Key | 输入参数。与 LM Studio 身份验证的 API 密钥。 |
| temperature | Model Temperature | 输入参数。模型的温度设置。 |
