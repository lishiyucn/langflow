title: 加入 Langflow 社区
slug: /contributing-community
---

您可以通过多种方式与 Langflow 社区互动并了解更多有关 Langflow 代码库的信息。

## 加入 Langflow Discord 服务器

加入 [Langflow Discord 服务器](https://discord.gg/EqksyE2EX9) 提问并展示您的项目。

## 在 X 上关注 Langflow

在 X 上关注 [@langflow_ai](https://twitter.com/langflow_ai) 以获取有关 Langflow 的最新消息。

## 在 GitHub 上为 Langflow 加星

如果您喜欢 Langflow，您可以为 [Langflow GitHub 仓库](https://github.com/langflow-ai/langflow) 加星。
星标帮助其他用户更容易找到 Langflow，并快速了解其他用户认为它有用。

因为 Langflow 是一个开源项目，仓库越可见，项目越有可能吸引[贡献者](/contributing-how-to-contribute)。

## 关注 GitHub 仓库

您可以关注 [Langflow GitHub 仓库](https://github.com/langflow-ai/langflow) 以获得有关新版本和其他仓库活动的通知。

要仅获取发布通知，请选择**仅发布**。

如果您选择**关注**，您将收到有关新版本以及问题、讨论和拉取请求的通知（如果您对这些活动感兴趣）。
有关自定义仓库通知的信息，请参阅 [GitHub 仓库订阅文档](https://docs.github.com/en/account-and-profile/managing-subscriptions-and-notifications-on-github/managing-subscriptions-for-activity-on-github/viewing-your-subscriptions)。

## 通过 GitHub 请求增强和获得帮助

您还可以通过 GitHub 仓库提交功能请求并获得有关 Langflow 的帮助。
有关更多信息，请参阅[获取帮助和请求增强](/contributing-github-issues)。
