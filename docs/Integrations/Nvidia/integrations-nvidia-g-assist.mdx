:::important
此组件仅适用于在 Windows 系统上使用 NVIDIA GPU 的 Langflow 用户。
:::

**NVIDIA G-Assist** 组件能够通过自然语言提示与 NVIDIA GPU 驱动程序进行交互。

例如，向 G-Assist 提问 `"我当前的 GPU 温度是多少？"` 或 `"显示可用的 GPU 内存"` 来获取信息，然后告诉 G-Assist 修改您的 GPU 设置。

有关更多信息，请参阅 [NVIDIA G-assist 项目仓库](https://github.com/NVIDIA/g-assist)。

## 先决条件

* Windows 操作系统
* NVIDIA GPU
* 已安装 `gassist.rise` 包。此包已随 Langflow 一起安装。

## 在流程中使用 G-Assist 组件
1. 创建一个包含 **Chat input** 组件、**G-Assist** 组件和 **Chat output** 组件的流程。
2. 将 **Chat input** 组件连接到 **G-Assist** 组件的 **Prompt** 输入，然后将 **G-Assist** 组件的输出连接到 **Chat output** 组件。
3. 打开 **Playground**，然后询问关于您的 GPU 的问题。例如，"我当前的 GPU 温度是多少？"。
**G-Assist** 组件查询您的 GPU，响应将出现在 **Playground** 中。

### 输入

**NVIDIA G-Assist** 组件接受单个输入：
- `prompt`: 由 NVIDIA G-Assist 处理的人类可读提示。

### 输出

**NVIDIA G-Assist** 组件输出一个 [Message](/data-types#message) 对象，其中包含：
- `text`: NVIDIA G-Assist 的响应，包含完成的操作结果。
- NVIDIA G-Assist 消息响应被包装在 Langflow [Message](/data-types#message) 对象中。
